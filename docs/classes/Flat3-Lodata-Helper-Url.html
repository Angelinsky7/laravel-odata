Build a URL.

public static http_build_url(mixed $url[, mixed $parts = array() ][, int $flags = self::HTTP_URL_REPLACE ][, array<string|int, mixed> &$new_url = array() ]) : string

The parts of the second URL will be merged into the first according to the flags argument.

Parameters
$url : mixed

(part(s) of) an URL in form of a string or associative array like parse_url() returns

$parts : mixed = array()

same as the first argument

$flags : int = self::HTTP_URL_REPLACE

a bitmask of binary or'ed HTTP_URL constants; HTTP_URL_REPLACE is the default

$new_url : array<string|int, mixed> = array()

if set, it will be filled with the parts of the composed url like parse_url() would return
